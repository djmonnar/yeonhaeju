01 · GETTING STARTED
앱 열기 / 기본 화면
앱 접속 방법
1
링크 접속
담당자에게 받은 링크를 스마트폰 브라우저에서 열어주세요. 크롬(Chrome) 또는 사파리(Safari) 모두 가능합니다.
2
홈 화면에 추가 (권장)
브라우저 메뉴 → 홈 화면에 추가를 누르면 앱처럼 바탕화면에 아이콘이 생깁니다. 다음부터는 아이콘만 누르면 바로 열립니다.
화면 구성
◈ 홈 ◉ 통계 ◇ 달력 ✦ 입력 🍶 사케 ≡ 더보기

화면 하단에 6개 탭이 있습니다. 원하는 탭을 터치해서 이동하세요.

💡 테마 변경
화면 우측 상단 ☀ 라이트 버튼을 누르면 밝은 화면으로 바꿀 수 있습니다. 다시 누르면 어두운 화면으로 돌아옵니다.

02 · DAILY ENTRY
매출 입력 방법
입력 순서
1
하단 ✦ 입력 탭 터치
날짜는 자동으로 오늘 날짜가 들어갑니다. 다른 날짜를 입력할 때는 날짜를 직접 바꿔주세요.
2
총 매출 입력
그날의 총 매출 금액을 입력합니다. 월 누적 금액도 함께 입력해 주세요.
3
결제수단별 입력
카드, 계좌이체, 제로페이, 현금을 각각 입력합니다.
합계가 총 매출과 다르면 빨간 경고와 차액이 표시되니 확인해주세요.
4
층별 / 시간대 입력 (자동 계산)
1층이나 2층 중 한 쪽만 입력하면 나머지가 자동으로 계산됩니다. 점심/저녁도 마찬가지입니다.
5
선결제 · 사케 · 메모 입력
선결제가 있으면 마이너스 금액으로 입력합니다 (예: -158000). 판매한 사케는 목록에서 선택 후 수량 입력. 특이사항은 메모란에 자유롭게 작성하세요.
6
✦ 저장하기 버튼 터치
저장되면 자동으로 다음 날짜로 넘어갑니다. 상단에 동기화됨 표시가 나타나면 저장 완료입니다.
⚠ 결제수단 차액 알림
카드 + 계좌이체 + 제로페이 + 현금의 합계가 총 매출과 다를 경우, 입력 도중 빨간 경고창이 나타납니다. 차액도 함께 표시되니 잘못 입력한 항목을 바로 확인할 수 있습니다.
💡 이미 입력한 날짜를 수정하려면, 달력에서 해당 날짜를 터치한 뒤 수정하기 →를 누르면 됩니다.
◇
03 · CALENDAR VIEW
달력으로 조회하는 법
특정 날짜 조회
1
하단 ◇ 달력 탭 터치
매출이 입력된 날짜는 숫자 아래에 금색 점이 표시됩니다.
2
날짜 터치
날짜를 터치하면 아래에 해당 날의 매출 상세 내역이 바로 펼쳐집니다. 총매출, 카드, 점심/저녁, 사케 등을 한눈에 확인할 수 있습니다.
기간 합계 조회 (특정일 ~ 특정일)
1
시작 날짜 터치
조회하고 싶은 기간의 첫 번째 날짜를 터치합니다. 날짜가 금색으로 선택됩니다.
2
종료 날짜 터치
두 번째 날짜를 터치하면 기간이 선택되고, 두 날짜 사이가 연한 금색으로 표시됩니다.
3
기간 합계 확인
달력 아래에 선택 기간의 총 매출, 일 평균, 카드, 점심/저녁, 사케 판매량이 자동으로 계산되어 표시됩니다.
✕ 기간 선택을 취소하려면 날짜 옆의 ✕ 버튼을 누르세요.
◉
04 · STATISTICS
통계 보는 법
4가지 기간 단위
📅
일별
특정 날짜 하루 상세 + 전년 동일일 비교
📆
주별
한 주 합계 + 요일별 매출 바 차트
🗓
월별
한 달 합계 + 전년 동월 비교 + 일별 차트
📊
연별
연간 총 매출 + 전년 비교 + 월별 차트
사용 방법
1
하단 ◉ 통계 탭 터치
2
일 / 주 / 월 / 연 탭 선택
보고 싶은 기간 단위를 선택합니다.
3
날짜 또는 월 선택
달력 아이콘을 눌러 원하는 날짜/월/연도를 선택하면 해당 기간 통계가 바로 표시됩니다.
📈 전년 비교
월별·연별 통계에는 전년도 같은 기간과 자동으로 비교해 ▲상승 또는 ▼하락 퍼센트가 표시됩니다.
🍶
05 · SAKE MANAGEMENT
사케 관리 방법
사케 품목 등록
1
하단 🍶 사케 탭 터치
2
우측 상단 + 등록 버튼 터치
이름, 종류, 용량, 초기 재고 수량을 입력합니다.
알림 기준 재고를 설정하면 재고가 그 이하로 떨어질 때 빨간색으로 표시됩니다.
재고 수정
1
해당 사케의 ✎ 버튼 터치
납품이 들어왔거나 재고를 직접 조정할 때 사용합니다.
2
현재 수량 입력 후 저장
매출 입력 시 판매 기록
1
입력 탭 → 사케 판매 기록 섹션
등록된 사케 목록이 드롭다운으로 표시됩니다. 선택 후 판매 수량을 입력합니다.
2
여러 종류 판매 시
+ 사케 추가 버튼으로 항목을 늘려서 입력할 수 있습니다.
📊 자동 통계
사케 탭 하단에 이달 품목별 판매량과 월별 판매 추이가 자동으로 차트로 표시됩니다.
🔄
06 · FIREBASE SYNC
실시간 동기화
동기화 상태 확인
●
동기화됨
정상. 데이터가 저장되어 있습니다.
●
저장 중
잠시 기다려 주세요.
●
오류
인터넷 연결을 확인하고 페이지를 새로고침해 주세요.
✅ 3명이 동시에 확인 가능
사장님, 팀장, 직원 모두 같은 링크로 접속하면 동일한 데이터를 실시간으로 볼 수 있습니다. 한 사람이 입력하면 다른 사람 화면에도 바로 반영됩니다.
📶 인터넷이 연결되어 있어야 데이터가 저장되고 동기화됩니다. 와이파이 또는 모바일 데이터가 필요합니다.
🔒 데이터는 Google Firebase 서버에 안전하게 저장됩니다. 브라우저를 지우거나 기기를 바꿔도 데이터는 사라지지 않습니다.
